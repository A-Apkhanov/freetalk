# [FreeTalk](https://freetalk.vercel.app/) 

### Freetalk помогает улучшить свои знания английского языка ### 

**Приложение включает в себя следующие функции:**
* Расширенный перевод
* Онлайн словарь
* Тренировки
* Коллекции


**Технологический стек:**
* React
* Redux
* FireBase
* TypeScript
* Ant Design
